# 🚀 快速操作指南 - v5.5

## ⚡ 3步完成（10分钟）

### 第1步：编译烧录固件 ⏱️ 3分钟

```
1. 打开Keil MDK
   双击：firmware/Tamlate.uvprojx

2. 清理编译
   Project → Clean Targets
   Project → Build (F7)
   
   ✅ 确认：0 Error(s), 0 Warning(s)

3. 烧录
   Flash → Download (F8)
   
   ✅ 确认：Programming Done

4. 复位GD32
   按RESET按钮
```

### 第2步：启动Web界面 ⏱️ 1分钟

```bash
cd react-bode-analyzer
npm run dev

# 或双击
./启动应用.sh

# 打开浏览器
http://localhost:5173
```

### 第3步：测试扫频 ⏱️ 5分钟

```
1. 点击"连接串口"
2. 选择GD32串口
3. 点击"启动扫频"
4. 等待40秒
5. 查看相位曲线是否平滑！
```

## ❓ 如果有问题

### 相位仍有跳变？
```
1. 确认串口显示 "Phase Unwrapping: Enabled"
2. 重新烧录固件
3. GD32按RESET复位
4. 浏览器强制刷新（Cmd+Shift+R）
```

### 编译错误？
```
1. Clean项目
2. 重新Build
3. 检查Keil版本（推荐5.0+）
```
